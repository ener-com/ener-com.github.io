# ☀️ Paneles Solares Enercom

Sitio web profesional de Paneles Solares Enercom — Servicio de instalación de paneles solares en México.

## 🛠️ Stack Tecnológico

- **HTML5** — Estructura semántica
- **Tailwind CSS** (CDN) — Diseño responsive y moderno
- **Vanilla JavaScript** — Interactividad sin dependencias
- **Formspree** — Procesamiento de formularios de contacto
- **GitHub Pages** — Hosting estático gratuito

## 🚀 Despliegue

Tu sitio se publica gratis en `https://<TU-USUARIO>.github.io` usando GitHub Pages.

Consulta la guía completa en [DEPLOYMENT.md](./DEPLOYMENT.md).

## 📋 Características

- ✅ Diseño responsive (móvil, tablet, desktop)
- ✅ Botón sticky de WhatsApp con Click-to-Chat API
- ✅ Formulario de contacto con Formspree
- ✅ Animaciones de scroll reveal
- ✅ Contadores animados
- ✅ Placeholders para Google Analytics 4 y Meta Pixel
- ✅ SEO básico con Open Graph tags
- ✅ Optimizado para GitHub Pages
